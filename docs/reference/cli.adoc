= CLI Reference

[source,bash]
----
adocserver [docsDir] [options]
----

`docsDir` is the path to your docs directory. Defaults to the current working directory.

== Options

[%header,cols="2,3,1"]
|===
|Flag |Description |Default

|`--port <n>`
|Port to listen on
|`3000`

|`--host <h>`
|Host/interface to bind
|`0.0.0.0`

|`--open`
|Open the browser automatically on start
|off

|`--help`, `-h`
|Print usage and exit
|
|===

== Examples

[source,bash]
----
# Serve docs/ in the current project
adocserver ./docs

# Different port, open browser automatically
adocserver ./docs --port 4000 --open

# Restrict to localhost only
adocserver ./docs --host localhost

# Serve from the current directory
adocserver
----

== Exit codes

[%header,cols="1,3"]
|===
|Code |Reason

|`0`
|Clean exit

|`1`
|`docsDir` does not exist, or `index.adoc` is missing from it
|===
